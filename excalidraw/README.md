# Diagramas Excalidraw

Esta pasta contém diagramas técnicos criados com o [Excalidraw](https://excalidraw.com/).

## Arquivos

- **arquitetura_cobranca_v1.excalidraw**: Diagrama da versão 1 da arquitetura de cobrança.
- **cls_primeiro_fluxo.excalidraw**: Diagrama representando um fluxo inicial (CLS).
- **zamp_brazil.excalidraw**: Diagrama relacionado ao projeto/contexto Zamp Brazil.

## Como visualizar

Para visualizar ou editar estes arquivos, você pode:
1. Carregá-los diretamente em [excalidraw.com](https://excalidraw.com/).
2. Usar uma extensão do VS Code como a "Excalidraw" (pomdtr.excalidraw-editor).
